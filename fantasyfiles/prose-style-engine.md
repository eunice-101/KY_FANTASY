# 문체·산문 스타일 엔진 (Prose Style Engine)

판타지 소설의 산문 품질을 베스트셀러 수준으로 끌어올리는 전문 도구. 장르별 문체 프로필을 설정하고, 초안의 산문을 다각도로 진단하여 개선한다.

이 스킬의 핵심 철학은 **"문체는 장식이 아니라 서사 전략이다"**이다. 같은 이야기도 에픽 판타지의 웅장한 산문과 어반 판타지의 날카로운 산문에서는 완전히 다른 경험이 된다. Ben Blatt의 *Nabokov's Favorite Word Is Mauve*(2017)에서 밝혀진 바와 같이, 베스트셀러 소설의 산문에는 통계적으로 확인 가능한 패턴이 존재한다.

---

## 전체 흐름

```
Mode A: 문체 프로필 설정 (집필 전)
Mode B: 산문 진단 (집필 후)
Mode C: 산문 개선 (진단 기반 리라이팅)
```

사용자의 상태에 따라 적절한 모드로 진입한다.
- 소설 시작 전 → Mode A부터
- 초안이 있음 → Mode B부터
- 진단 결과가 있음 → Mode C부터

---

## Mode A: 문체 프로필 설정

집필을 시작하기 전에, 이 소설의 산문이 어떤 느낌이어야 하는지 결정한다.

### 수집 항목

사용자에게 다음을 질문하거나 함께 결정한다:

| 항목 | 질문 | 선택지 |
|------|------|--------|
| **장르** | 어떤 판타지 하위 장르인가? | 에픽/하이, 어반, 다크, YA, 로맨스, 포털/이세계, 웹소설 |
| **참조 작가** | 문체를 참고할 작가가 있는가? | 톨킨, 샌더슨, 롤링, 르 귄, 마틴, 로스퍼스 등 |
| **독자층** | 주 독자의 연령대와 독서 경험은? | 10대, 20~30대, 40대+, 장르 매니아, 일반 독자 |
| **톤** | 분위기는? | 웅장, 어둡고 건조, 유머러스, 서정적, 긴박, 몽환적 |
| **언어** | 한국어? 영어? | 한국어 출판, 한국어 웹소설, 영어 |

### 장르별 문체 프로필 기본값

아래는 기본값이다. 사용자의 선택에 따라 조정한다.

**에픽/하이 판타지** (톨킨, 샌더슨, 조던 스타일)
- 대화 비율: 25~35%
- 평균 문장 길이: 18~28단어 (한국어 기준 30~50자)
- 묘사 밀도: 높음 — 환경·전투·마법 장면에서 감각 묘사 4~5가지 이상
- 어휘 수준: 중상 — 고어체나 아르카익한 표현 허용, 단 과도한 장식 지양
- 서술 비율: 내면 독백 15~20%, 액션/이벤트 30~35%, 묘사 20~25%
- 특징: 긴 문단, 점층적 서술, 시적 비유 허용

**어반 판타지** (짐 버처, 닐 게이먼 스타일)
- 대화 비율: 40~55%
- 평균 문장 길이: 12~20단어 (한국어 기준 20~35자)
- 묘사 밀도: 중간 — 핵심 감각 2~3가지, 빠르게 제시
- 어휘 수준: 중 — 구어적, 위트 있는 표현, 현대적 비유
- 서술 비율: 내면 독백 10~15%, 액션/이벤트 35~45%, 묘사 10~15%
- 특징: 짧은 문단, 1인칭 화자의 독특한 목소리, 유머와 아이러니

**다크 판타지** (조 애버크롬비, 마크 로렌스 스타일)
- 대화 비율: 30~40%
- 평균 문장 길이: 10~22단어 (한국어 기준 18~40자)
- 묘사 밀도: 선택적 고밀도 — 잔혹한 장면은 세밀하게, 나머지는 건조하게
- 어휘 수준: 중 — 건조하고 직접적, 미사여구 최소
- 서술 비율: 내면 독백 20~25%, 액션/이벤트 30~35%, 묘사 15~20%
- 특징: 불편한 진실을 직시하는 톤, 아이러니, 도덕적 회색지대

**YA 판타지** (롤링, 수잔 콜린스 스타일)
- 대화 비율: 35~50%
- 평균 문장 길이: 12~20단어 (한국어 기준 20~35자)
- 묘사 밀도: 중간 — 감각 묘사 3가지, 감정 중심
- 어휘 수준: 중하 — 접근성 우선, 전문 용어는 즉시 설명
- 서술 비율: 내면 독백 15~25%, 액션/이벤트 30~40%, 묘사 10~15%
- 특징: 감정적 몰입, 또래 관계 중심 대화, 성장 내러티브

**한국 웹소설 판타지**
- 대화 비율: 45~60%
- 평균 문장 길이: 8~18단어 (한국어 기준 15~30자)
- 묘사 밀도: 낮음~중간 — 액션 묘사는 짧고 강렬하게
- 어휘 수준: 중하 — 가독성 최우선, 한 줄에 하나의 정보
- 서술 비율: 내면 독백 10~15%, 액션/이벤트 40~50%, 묘사 5~10%
- 특징: 짧은 문단(2~3문장), 빈번한 줄바꿈, 독자와의 직접 소통감

### 산출물

문체 프로필 문서를 생성한다. 대화창에 요약 제시 후, 사용자 확인을 받는다. 복잡한 설정은 `prose-profile.md`로 저장한다.

## Mode B: 산문 진단

초안 텍스트를 입력받아 6가지 관점에서 산문 품질을 분석한다.

### 진단 1: 대화-서술 균형 분석

설정된 장르 프로필의 대화 비율 기준과 실제 텍스트를 비교한다.

점검 항목:
- 전체 대화 비율이 장르 기준 범위 내인가?
- 챕터별/장면별 대화 비율 편차가 30% 이상 벌어지는 구간이 있는가?
- 대화 없이 서술만 이어지는 구간이 3페이지(약 1,500자) 이상 지속되는가?
- 대화만으로 이루어진 구간이 2페이지(약 1,000자) 이상 지속되는가?

출력: 각 항목에 대해 [적정 / 조정 필요]를 판정하고, 조정 필요 시 구체적 위치와 방향을 제시한다.

### 진단 2: 문장 리듬 분석

문장 길이의 변화가 자연스러운 리듬을 만드는지 점검한다.

점검 항목:
- 같은 길이의 문장이 5개 이상 연속되는 구간이 있는가? (단조로움 경고)
- 긴장감 높은 장면에서 문장이 충분히 짧아지는가? (10단어 이하)
- 이완 장면에서 문장이 자연스럽게 길어지는가?
- 문단 시작 문장이 다양한 구조를 갖는가? (주어로 시작하는 문장이 80% 이상이면 경고)

출력: 문장 길이 분포도(히스토그램 형태의 요약)와 문제 구간을 표시한다.

### 진단 3: 감각 묘사 밀도 점검

장면별로 사용된 감각의 종류와 빈도를 분석한다.

점검 항목:
- 시각 편향: 시각 묘사가 전체 감각 묘사의 70% 이상을 차지하는가?
- 장면당 감각 종류: 각 장면에 최소 3가지 감각이 포함되었는가?
- 마법 장면의 고유 감각: 마법 사용 장면에 물리적 감각(열, 전류, 압력 등)이 포함되었는가?
- 음식·장소 장면의 후각/미각: 식사 장면이나 새로운 장소 등장 시 해당 감각이 있는가?

출력: 감각 유형별 사용 빈도와 부족한 감각을 장면별로 표시한다.

### 진단 4: 동사 강도 분석

약한 동사(be동사, ~하다, ~이다, ~있다)의 과다 사용을 점검하고 대안을 제시한다.

점검 항목:
- be동사/이다/있다 비율이 전체 동사의 30%를 초과하는가?
- 액션 장면에서 구체적이고 강렬한 동사가 사용되었는가?
- 같은 동사가 한 문단 내에서 반복되지 않는가?
- 동사의 시제가 일관되는가? (과거형 서술 중 갑작스러운 현재형 전환 탐지)

출력: 약한 동사 목록과 각각에 대한 강한 대체 동사 제안.

### 진단 5: 클리셰 탐지

판타지 소설에서 흔히 발견되는 진부한 표현과 비유를 탐지한다.

탐지 대상 카테고리:
- **감정 클리셰**: "심장이 뛰었다", "숨이 멎었다", "피가 끓었다", "눈물이 흘렀다" 등 — 해당 감정을 더 독창적으로 표현할 방법을 제안한다
- **환경 클리셰**: "달빛이 비추었다", "바람이 불었다", "구름이 몰려왔다" — 이 세계관 고유의 환경 묘사로 전환할 방법을 제안한다
- **전투 클리셰**: "검이 번뜩였다", "피가 튀었다", "전장에 고요가 내려앉았다" — 캐릭터의 신체 감각과 심리 상태 중심으로 전환할 방법을 제안한다
- **마법 클리셰**: "빛이 번쩍였다", "에너지가 폭발했다" — 이 소설의 마법 체계에 맞는 고유 묘사로 전환할 방법을 제안한다

출력: 탐지된 클리셰 목록 + 각각에 대한 2~3가지 대체 표현 제안.

### 진단 6: Info-dump 탐지

세계관 정보나 설정이 이야기 흐름을 방해하는 방식으로 전달되는 구간을 찾는다.

탐지 기준:
- 서술(대화 없는 설명)이 500자 이상 이어지면서 플롯을 진전시키지 않는 구간
- 두 캐릭터가 서로 이미 알고 있는 정보를 대화로 주고받는 "As You Know, Bob" 패턴
- 역사/배경 설명이 현재 장면의 감정이나 행동과 연결되지 않는 구간

출력: 해당 구간의 위치 + "이 정보를 캐릭터의 행동/대화/발견을 통해 전달하는 방법" 제안.

### 진단 결과 출력 형식

모든 진단 결과를 아래 형식으로 통합 제시한다:

```
## 산문 진단 리포트

### 전체 요약
- 장르 프로필: [설정된 장르]
- 전체 대화 비율: XX% (기준: XX~XX%)
- 평균 문장 길이: XX단어 (기준: XX~XX)
- 감각 묘사 다양성: X/5 감각 활용
- 동사 강도 점수: XX/100
- 클리셰 발견: X건
- Info-dump 의심: X건

### 상세 진단
[진단 1~6의 상세 결과]

### 우선 개선 항목 (Top 3)
1. [가장 시급한 문제]
2. [두 번째 시급한 문제]
3. [세 번째 시급한 문제]
```

사용자 확인 후 Mode C로 진행한다.

## Mode C: 산문 개선

진단 결과를 바탕으로 실제 텍스트를 수정한다.

### 개선 원칙

- **의미 보존**: 원문의 서사적 의미와 감정적 의도를 변경하지 않는다
- **최소 개입**: 문제가 없는 문장은 건드리지 않는다
- **장르 적합성**: 모든 수정은 설정된 장르 프로필에 부합해야 한다
- **작가 존중**: 작가의 고유 표현이 의도적이라면 유지한다. 판단이 어려우면 사용자에게 확인한다

### 개선 순서

1. Info-dump 해소 (구조적 문제 우선)
2. 클리셰 교체 (독창성 향상)
3. 동사 강화 (문장 활력)
4. 감각 묘사 보강 (몰입감)
5. 문장 리듬 조정 (가독성)
6. 대화-서술 균형 조정 (페이싱)

### 산출물

수정본은 대화창에서 제시하되, 분량이 길면 `prose-revised.md`로 저장한다. 변경 사항 요약을 반드시 포함한다.

## 스킬 연계

### fantasy-novel-writer와의 연동

- Step 5(챕터 집필) 시작 시: Mode A의 문체 프로필이 설정되어 있으면 해당 프로필을 자동 적용한다
- Step 6(리라이팅) 시작 시: Mode B의 산문 진단을 리라이팅 점검에 통합한다

### dialogue-voice-engine과의 연동

- 대화 비율 분석(진단 1)의 결과를 dialogue-voice-engine에 전달하여, 대화 품질 개선과 분량 조정을 연계한다
