# 대화 음성 분화 엔진 (Dialogue Voice Engine)

판타지 소설에서 5~15명의 캐릭터가 각각 즉시 구분 가능한 고유 화법을 갖도록 설계하고, 집필 과정에서 일관되게 유지하며, 대화 품질을 진단하는 전문 도구.

이 스킬의 핵심 철학은 **"대사만 보고 누가 말했는지 알 수 있어야 한다"**이다. J.K. 롤링의 해리포터에서 해그리드의 투박한 사투리, 덤블도어의 절제된 지혜, 스네이프의 차갑고 조소적인 말투가 서로 결코 혼동되지 않는 것처럼, 각 캐릭터에게 고유한 "음성 DNA"를 부여하는 것이 이 스킬의 목적이다.

---

## 전체 흐름

```
Mode A: 음성 DNA 설계 (캐릭터 생성 시)
Mode B: 대화 생성 (집필 시)
Mode C: 대화 진단 (리라이팅 시)
```

---

## Mode A: 음성 DNA 설계

각 주요 캐릭터에게 고유한 화법 프로필(음성 DNA)을 만든다.

### 음성 DNA 구성 요소

캐릭터당 아래 7가지 차원을 설정한다. 모든 차원은 스펙트럼 위의 위치로 정의한다.

**1. 어휘 레벨** (Vocabulary Level)
- 스펙트럼: 원초적 ←——→ 학술적
- 설정 예시: 해그리드=원초적(2/10), 덤블도어=학술적(9/10), 해리=중간(5/10)
- 이 캐릭터가 쓰는 단어의 복잡도를 결정한다
- 교육 수준, 사회적 배경, 직업에 따라 결정한다

**2. 문장 호흡** (Sentence Rhythm)
- 스펙트럼: 끊기는 짧은 호흡 ←——→ 길고 유려한 호흡
- 설정 예시: 충동적 전사=짧은(2/10), 현자=긴(8/10), 교활한 정치가=중간+변화(5/10, 고변동)
- 성격에 따라 결정한다: 충동적·분노형=짧음, 지적·성찰형=길음, 교활·계산형=상황에 따라 변화

**3. 습관어/말버릇** (Verbal Tics)
- 캐릭터를 즉시 식별할 수 있는 고유 표현 1~3개
- 설정 예시: "솔직히 말하자면..." / "~란 말이지" / "흥미롭군" / "어쨌든"
- 주의: 너무 자주 사용하면 독자를 짜증나게 한다. 3~5페이지에 1번 정도가 적절하다
- 감정 상태에 따라 습관어가 변하면 더 효과적이다 (평소: "흥미롭군" → 위기 시: 습관어 사라짐)

**4. 감정 표현 방식** (Emotional Expression)
- 스펙트럼: 억제적(내면화) ←——→ 폭발적(외현화)
- 설정 예시: 스네이프=극도로 억제적(1/10), 론 위즐리=폭발적(8/10)
- 억제적 캐릭터: 감정을 행동과 신체 반응으로 표현 (주먹을 꽉 쥐었다, 시선을 피했다)
- 폭발적 캐릭터: 감정을 대사와 목소리 톤으로 직접 표현
- 캐릭터의 감정 표현 방식이 변하는 것 자체가 성장 아크의 지표가 될 수 있다

**5. 유머 스타일** (Humor Type)
- 유형: 빈정대기(sarcasm) / 말장난(wordplay) / 자기비하(self-deprecation) / 상황 유머(observational) / 건조(deadpan) / 유머 없음(none)
- 설정 예시: 프레드&조지=말장난+상황유머, 스네이프=빈정대기+건조, 볼드모트=유머없음
- 유머의 유무와 종류는 캐릭터의 지능, 사회성, 자기인식을 드러낸다

**6. 대화 전략** (Conversational Strategy)
- 유형: 직진형(direct) / 우회형(indirect) / 질문형(Socratic) / 명령형(commanding) / 회피형(evasive)
- 설정 예시: 해리=직진형, 덤블도어=질문형, 드레이코=명령형, 스네이프=우회형+빈정
- 같은 정보를 전달할 때도 전략에 따라 완전히 다른 대사가 나온다
- 한 캐릭터가 상대에 따라 전략을 바꾸면 관계의 역학을 보여줄 수 있다

**7. 사회언어학적 마커** (Sociolinguistic Markers)
- 방언/사투리 사용 여부와 정도
- 존댓말/반말 전환 패턴 (한국어 소설에서 특히 중요)
- 직업·계층 전문 용어 (기사의 전투 용어, 마법사의 주문 용어, 상인의 거래 용어)
- 세대 차이를 반영하는 표현 (고대 존재는 고풍스러운 표현, 젊은 캐릭터는 현대적 어감)

### 음성 DNA 카드 출력 형식

각 캐릭터의 음성 DNA를 카드 형식으로 출력한다:

```
═══════════════════════════════════
[캐릭터 이름] — 음성 DNA 카드
═══════════════════════════════════
어휘 레벨:    ■■■■■□□□□□  (5/10) 중간
문장 호흡:    ■■■□□□□□□□  (3/10) 짧은 편
습관어:       "솔직히" / "~란 말이지"
감정 표현:    ■■■■■■■■□□  (8/10) 폭발적
유머:         자기비하 + 상황유머
대화 전략:    직진형 (위기 시 → 회피형)
사회언어:     비격식, 친구에게 반말, 어른에게 존댓말
───────────────────────────────────
샘플 대사 (평상시):
  "아, 솔직히 이거 나한테 왜 떠넘기는 건데?
   누가 봐도 [조력자 이름]이 적임자란 말이지."
샘플 대사 (위기 시):
  "...됐어. 내가 갈게."
═══════════════════════════════════
```

### 교차 구분 테스트

모든 주요 캐릭터의 음성 DNA를 설정한 뒤, 반드시 **교차 구분 테스트**를 실행한다.

테스트 방법:
1. 동일한 상황(예: "적의 습격을 보고하는 장면")을 설정한다
2. 각 캐릭터가 이 상황에서 할 대사를 1~2문장씩 생성한다
3. 대사만 보고 어떤 캐릭터인지 구분할 수 있는지 확인한다
4. 구분이 어려운 캐릭터 쌍이 있으면, 7가지 차원 중 최소 2가지를 더 차별화한다

### 산출물

음성 DNA 문서: `voice-dna.md`. 모든 주요 캐릭터의 카드 + 교차 구분 테스트 결과 포함.

## Mode B: 대화 생성

설정된 음성 DNA를 바탕으로 장면별 대화를 작성한다.

### 장면별 대화 작성 체크리스트

대화를 쓸 때마다 다음을 점검한다:

**1. 목적 확인**: 이 대화가 서사적으로 달성하는 것은 무엇인가?
- 정보 전달 (세계관/플롯)
- 관계 변화 (친밀도, 갈등, 배신)
- 캐릭터 드러내기 (성격, 비밀, 감정)
- 긴장 고조 또는 유머를 통한 이완

**2. 서브텍스트 설계**: 캐릭터가 "말하는 것"과 "의미하는 것"이 다른가?
- 좋은 대화는 표면(말하는 것)과 이면(의미하는 것) 두 층위를 갖는다
- 서브텍스트가 있으면 독자는 "행간을 읽는" 쾌감을 느낀다
- 모든 대화에 서브텍스트가 필요한 것은 아니다. 중요한 장면에 집중한다

**3. 대화 태그 전략**: "~라고 말했다"를 기본으로 사용한다
- 대화 태그(said/말했다)는 투명해야 한다. 독자가 의식하지 않아야 한다
- "외쳤다", "속삭였다", "절규했다" 등의 변형은 한 챕터에 2~3회 이내로 제한한다
- 대화 태그 대신 액션 비트(행동 묘사)를 사용하면 장면이 살아난다:
  - 나쁜 예: "가지 마!" 그녀가 외쳤다.
  - 좋은 예: "가지 마!" 그녀가 그의 소매를 움켜쥐었다.

**4. 비트 배치**: 대화 사이에 액션 비트와 내면 독백을 적절히 배치한다
- 빠른 대화 교환 → 비트 최소화 (긴장감)
- 감정적 대화 → 내면 독백 비트 삽입 (깊이)
- 정보 전달 대화 → 환경 묘사 비트 삽입 (Info-dump 방지)

### "As You Know, Bob" 방지 규칙

캐릭터가 서로 이미 아는 정보를 대화로 주고받으면 안 된다. 대신:
- 새로 합류한 캐릭터에게 설명하는 형태로 정보 전달
- 기존 정보에 새로운 의미를 부여하는 형태 ("그때 그건 알고 보니...")
- 의견 충돌을 통해 정보를 자연스럽게 표출 ("넌 그렇게 생각하지? 난 달라")
- 캐릭터가 잘못 알고 있는 정보를 교정하는 형태

## Mode C: 대화 진단

작성된 대화의 품질을 점검하고 개선안을 제시한다.

### 진단 1: 음성 일관성 검증

각 캐릭터의 대사가 음성 DNA 카드와 일치하는지 점검한다.

점검 항목:
- 어휘 레벨 이탈: 교육 수준이 낮은 캐릭터가 갑자기 학술적 단어를 사용하지 않는가?
- 문장 호흡 이탈: 짧은 호흡 캐릭터가 갑자기 긴 복문을 사용하지 않는가?
- 습관어 빈도: 너무 자주(매 페이지) 또는 너무 드물게(10페이지 이상 없음) 나오지 않는가?
- 감정 표현 이탈: 억제적 캐릭터가 갑자기 폭발하거나, 폭발적 캐릭터가 갑자기 침묵하지 않는가? (의도적인 경우는 제외 — 이것이 성장 포인트가 될 수 있다)

출력: 이탈 발견 시 해당 대사와 음성 DNA 카드를 비교하여 수정 방향을 제안한다. 의도적 이탈이 의심되면 사용자에게 확인한다.

### 진단 2: 교차 구분 테스트 (실전)

실제 원고에서 무작위로 대사 10개를 추출하고, 대사만 보고 캐릭터를 식별할 수 있는지 테스트한다.

테스트 방법:
1. 원고에서 대화 태그와 주변 서술을 제거한 대사만 추출
2. 각 대사가 어느 캐릭터의 것인지 추론
3. 정답률이 70% 미만이면, 구분이 안 되는 캐릭터 쌍을 식별하고 차별화 방안을 제시한다

### 진단 3: 서브텍스트 깊이 점검

중요한 대화 장면에 서브텍스트가 존재하는지 점검한다.

점검 항목:
- 핵심 갈등 장면에서 캐릭터가 직접적으로 자기 감정을 말하고 있지는 않은가? ("나 화났어"보다 "그래, 네가 알아서 해"가 더 풍부하다)
- 중요한 관계 변화 장면에서 말하지 않는 것이 말하는 것보다 더 많은가?
- 적대 캐릭터와의 대화에서 표면적 예의와 이면의 적대가 공존하는가?

### 진단 4: 대화 기능 분석

각 대화가 서사적 기능을 수행하는지 점검한다.

점검 항목:
- 플롯이나 캐릭터를 전혀 진전시키지 않는 장식적 대화가 있는가?
- 같은 정보를 다른 대화에서 이미 전달하고 있지는 않은가? (중복 체크)
- 대화 후 상황이나 관계에 변화가 있는가? (변화 없는 대화는 삭제 후보)

### 진단 결과 출력 형식

```
## 대화 진단 리포트

### 전체 요약
- 분석 대화 수: XX개 장면
- 음성 일관성: XX% (이탈 X건)
- 교차 구분 정답률: XX%
- 서브텍스트 포함 비율: XX% (핵심 장면 기준)
- 기능 없는 대화: X건

### 캐릭터별 음성 유지율
| 캐릭터 | 유지율 | 주요 이탈 유형 |
|--------|--------|---------------|
| ...    | ...    | ...           |

### 우선 개선 항목
1. [가장 시급한 문제]
2. [두 번째]
3. [세 번째]
```

## 스킬 연계

### fantasy-novel-writer와의 연동

- Step 3(캐릭터 창조) 완료 시: 각 캐릭터에 대해 Mode A를 실행하여 음성 DNA 카드를 생성한다
- Step 5(챕터 집필) 중: Mode B의 체크리스트를 대화 장면 작성에 적용한다
- Step 6(리라이팅) 중: Mode C의 진단을 리라이팅 점검에 통합한다

### prose-style-engine과의 연동

- 대화 비율 분석 결과를 받아, 대화가 너무 적으면 대화 장면 추가 제안, 너무 많으면 서술 보강 제안을 한다

- J.K. 롤링의 캐릭터 음성 분화 분석 (해그리드, 덤블도어, 스네이프의 화법 패턴)
