# 캐릭터 상태 — Ch 1 완료 후

## 서윤아 (주인공)
- **위치:** 한양 변두리, 집 마당
- **천서력:** 0단계 (잠재력. 글자 쓸 때 손끝에서 미세한 빛 — 본인 미인지)
- **감정 상태:** 호기심 + 불안. 할머니의 피, 서책 검열 소문, "보여줄 것"에 마음이 동요
- **보유 정보:** 서당 글 훔쳐 배움, 서책 검열 소문 인지, 할머니의 옥비녀 존재 인지
- **부상/컨디션:** 정상

## 서대비 (할머니)
- **위치:** 같은 집, 방 안
- **건강:** 기침에서 피 — 병세 심각 (과거 생술 사용 대가, 미공개)
- **감정 상태:** 전수 결심. 시간이 얼마 남지 않았음을 인지
- **보유 아이템:** 옥비녀
- **다음 행동:** 밤에 감나무 아래에서 윤아에게 비밀을 보여줄 예정

## 강도윤
- **위치:** 한양 시장
- **상태:** 평소. 윤아의 서당 도청을 알고 지지함
- **감정 상태:** 밝고 활기참. 과거 급제의 꿈

## 삼돌이
- **위치:** 한양 시장 떡집
- **상태:** 서책 검열 소문을 전달. 정보통 역할 확립
- **감정 상태:** 평소
