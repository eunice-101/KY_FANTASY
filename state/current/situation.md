# 상황 파일 — Ch 5 집필 직전

**챕터 번호:** 4 완료 → 5 집필 준비
**상태:** Ch 4 품질검증 통과, Ch 5 "천서각의 비밀" 집필 대기

## 직전 챕터 (Ch 4) 요약

이른 새벽, 이웃 아낙 경고로 포졸이 바로 윗 골목까지 수색함을 확인. 할머니 상태 악화 — "네가 결정해야 한다", "북한산에 가거라" 지시 후 기침 심화. 윤아가 보자기(두루마리+옥비녀)를 들고 도윤과 함께 도성 탈출. 북한산 중턱에서 귀 뒤의 점 발열(복선 #1 회수) — 나침반처럼 결계 위치 탐지. 이끼 낀 바위 삼각형에서 결계 발견. 도윤의 손이 막힘 확인. 옥비녀로 결계 개방(복선 #2 1차 회수). 천서각 석실 진입 — 빈 홈들, 석대, 고구려 수호 문양(복선 #7 설치), 기록 목록에서 '서씨' 두 글자 발견. 결계가 내부에서 소리 없이 닫힘. 도윤의 탈출 시도 실패. 벽에서 별도 통로가 열리며 미지 인물 등장. 훅: "기다렸다. 서씨 가문의 아이."

## 현재 열린 질문

- 미지 인물은 누구인가? 적인가 아군인가? → Ch 5에서 일부 공개
- 천서각의 기록 목록은 무엇인가? 서씨가 마지막인 이유는?
- 석대 옆면 비녀 굵기 홈 — 꽂으면 무슨 일이 생기는가? → Ch 5에서 사용 예정
- 빈 홈들에 있던 것들은 어디 갔는가?
- 할머니는 얼마나 버틸 수 있는가?
