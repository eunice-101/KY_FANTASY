# 상황 파일 — Ch 1 집필 직전

**챕터 번호:** 0 → 1 집필 준비 완료
**상태:** 기반 완료, Ch 1 "담 너머의 글" 집필 대기

## 소설 기본 상황

- **시대:** 조선 연산군 말기 (1506년 이전)
- **장소:** 한양
- **분위기:** 유교 사회의 억압 + 몰락한 양반가의 결핍 + 시장의 활기

## 주인공 현재 상태

서윤아(16세)는 갑자사화로 몰락한 양반가의 손녀. 할머니(서대비)와 한양 변두리 허름한 집에서 살고 있다. 여자라는 이유로 서당에 다닐 수 없어 담 너머에서 몰래 글을 배우고 있다. 천서력의 존재를 아직 모른다.
