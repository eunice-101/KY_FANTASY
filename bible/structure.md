# 소설 구조 계획 — 천서의 아이 (불변 — 생성 시작 후 수정 금지)

## 소설 기본 정보

- **제목:** 천서의 아이 (天書의 아이)
- **장르:** 조선 역사 판타지
- **총 챕터:** 25챕터
- **목표 분량:** 챕터당 5,000-8,000자 / 총 12.5-20만자

## 8막 구조

### 1막: 일상 세계 — "소녀는 공부하면 안 된다" (Ch 1-3)
- **감정 곡선:** 결핍 → 호기심 → 공포의 그림자
- Ch 1. 담 너머의 글
- Ch 2. 할머니의 상자
- Ch 3. 시장의 소문

### 2막: 모험의 부름 — "천서력을 깨우다" (Ch 4-6)
- **감정 곡선:** 모험 → 경이 → 절박함
- Ch 4. 북한산의 결계
- Ch 5. 천서각의 비밀
- Ch 6. 첫 번째 대가

### 3막: 문지기의 시험 — "배우고, 의심하고, 각성하다" (Ch 7-10)
- **감정 곡선:** 불안 → 좌절 → 성취 → 절박
- Ch 7. 임사홍의 제안
- Ch 8. 천서각의 수련
- Ch 9. 해월의 진심
- Ch 10. 동술의 각성

### 4막: 시련의 길 — "궁궐, 권력, 선택" (Ch 11-15)
- **감정 곡선:** 긴장 → 공포 → 액션 → 숨막힘
- Ch 11. 궁궐로 가는 길
- Ch 12. 연회의 밤
- Ch 13. 궁궐 지하의 봉인
- Ch 14. 추격
- Ch 15. 요녀 사냥

### 5막: 심연 — "잃어버린 모든 것" (Ch 16-18)
- **감정 곡선:** 배신 → 슬픔 → 경외
- Ch 16. 배신의 맛
- Ch 17. 북쪽으로
- Ch 18. 근원의 석실

### 6막: 보상과 귀환 — "진짜 힘의 의미" (Ch 19-20)
- **감정 곡선:** 각성 → 결속
- Ch 19. 돌아오는 길
- Ch 20. 재결합

### 7막: 최종 대결 — "반정의 밤" (Ch 21-24)
- **감정 곡선:** 긴장 → 카타르시스 → 감동 → 보상
- Ch 21. 반정의 밤 (1) — 궁문을 열어라
- Ch 22. 반정의 밤 (2) — 연산군과의 대면
- Ch 23. 반정의 밤 (3) — 피를 멈추다
- Ch 24. 새로운 왕

### 8막: 새로운 일상 — "천서의 아이" (Ch 25)
- **감정 곡선:** 따뜻한 감동 → 여운
- Ch 25. 천서의 아이

## 캐릭터 관계도

- 윤아 ↔ 할머니(서대비): 전승자와 후계자. 사랑하지만 기대가 짐
- 윤아 ↔ 도윤: 유일한 친구. 신분 차이를 넘은 우정, 열등감이 때때로 갈등
- 윤아 ↔ 해월: 처음 불신 → 정체 밝혀진 후 신뢰. 언니 같은 존재
- 윤아 ↔ 연산군: 폭군 vs 잠재적 도구. 가문 복권의 유혹
- 윤아 ↔ 임사홍: 가장 불안한 관계. 도움을 주지만 믿을 수 없다
- 도윤 ↔ 해월: 티격태격 → 묘한 존중
- 임사홍 ↔ 연산군: 표면적 충신, 서로를 이용하는 관계
