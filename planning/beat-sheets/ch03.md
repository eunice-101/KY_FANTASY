# Ch 3. 시장의 소문

## 막: 1막 — 일상 세계 (마지막)
## 감정 곡선: 일상의 따뜻함 → 검열의 서늘함 → 해월의 불안한 시선 → 천서력 발현의 공포 → 두려움의 그림자

## 핵심 사건
1. 이른 아침 할머니의 기침 악화 — "귀를 열어 두어라" 지시
2. 시장 — 서책 검열 칙령 게시, 포졸 등장 (복선 #5 설치)
3. 해월이 시장 비단 골목 경계에서 윤아를 관찰 — 귀 뒤 점 확인 (복선 #6 설치)
4. 도윤과 우물가 대화 — 신분 콤플렉스 직접 토로 (서브플롯 B 시작)
5. 밤에 혼자 두루마리 연습 — 의도치 않은 청백색 빛 발현, 촛불이 꺼짐
6. 훅: 도윤이 낮에 말했던 "북한산 청백색 빛" 회상 — 손끝의 빛과 같은 색

## 등장인물
- 서윤아
- 서대비 (할머니, 비트 1)
- 강도윤
- 삼돌이
- 해월 (이름 미공개로 첫 등장 — 기생/약장수 위장)

## 복선 설치
- #5 [물건] 서책 검열 칙령 → Ch15 회수 (요녀 사냥 — 검열이 실은 천서력 기록 수색)
- #6 [행동] 해월이 윤아의 귀 뒤 점을 시선으로 확인 → Ch7, Ch9 회수
- 부가: 복선 #1(귀 뒤 점) 시각적 연결, 복선 #3(대가) 사전 체감, 복선 #4(손끝의 빛) 재확인

## 챕터 훅
낮에 도윤이 스쳐 말했던 "북한산에서 청백색 빛을 봤다"가 밤의 방 안 어둠 속 윤아의 손끝에서 나왔다 사라진 빛과 겹친다. 같은 색이었다. 설명 없이 닫힌다.

## 집필 노트
- 시장 감각 묘사: 들기름 굽는 냄새/젓갈 비린내/흥정 소리/아이 울음/남색 치마
- 해월 첫 등장: 이름, 정체 절대 언급 금지. 시선의 방향(귀 뒤)만 정확하게
- 천서력 장면: 아름다움이 아닌 두려움 우선. 의도 없는 발현
- 촛불이 꺼지는 것: 윤아가 만든 것일 수 있다는 여지 남기기
- 훅: 도윤의 말을 낮에 심고 밤에 회수하는 지연 구조. "같은 색이었다" 세 글자로 닫기
- 분량 목표: 5,500-7,000자 (시장 묘사에 공들이되 밤 장면에서 긴장 집중)
