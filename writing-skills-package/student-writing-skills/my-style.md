# 나만의 글쓰기 스타일 (My Writing Style)

> 이 파일은 AI가 여러분의 글쓰기 스타일을 이해하기 위한 설정 파일입니다.
> 아래 `[여기에 작성]` 부분을 여러분만의 스타일로 채워넣어 주세요.
> 빈칸을 모두 채우지 않아도 됩니다. 채운 부분만 적용됩니다.

---

## 내 글쓰기 한 문장 정의

나는 어떤 글을 쓰고 싶은가? 한 문장으로 표현해보세요.

```
[여기에 작성]
예시: "어려운 개념을 쉽고 재미있게 풀어쓰는, 독자와 대화하는 글쓰기"
예시: "논리적이고 날카롭지만 따뜻한 시선이 느껴지는 글쓰기"
예시: "데이터와 사례로 설득하되, 유머를 잃지 않는 글쓰기"
```

---

## 어조 (Tone)

독자에게 어떤 느낌을 주고 싶은가?

```
[여기에 작성]
예시: "친근한 선배처럼 — 가르치지 않고 함께 탐색하는 느낌"
예시: "전문가이지만 딱딱하지 않은 — 커피 한 잔 하며 이야기하는 느낌"
예시: "분석적이고 객관적인 — 감정보다 논리로 설득하는 느낌"
```

## 문체 특징

### 좋아하는 문장 스타일

```
[여기에 작성]
예시: "짧고 임팩트 있는 문장 선호. 한 문장에 한 가지 정보만."
예시: "비유와 예시를 많이 활용. 추상적인 개념은 일상으로 번역."
예시: "질문을 던지며 독자의 생각을 이끌어내는 스타일."
```

### 싫어하는 문장 스타일

```
[여기에 작성]
예시: "~할 수 있다, ~일 것이다 같은 모호한 표현 싫어함"
예시: "딱딱한 논문체, 수동태 과다 사용 싫어함"
예시: "감정에 호소하는 문장, 과장된 형용사 싫어함"
```

## 나만의 무기 (Signature Move)

글에서 가장 자신 있는 부분이 있다면?

```
[여기에 작성]
예시: "비유 — 어떤 개념이든 일상의 비유로 바꿀 수 있다"
예시: "구조화 — 복잡한 정보를 깔끔하게 정리하는 것"
예시: "스토리텔링 — 개인 경험에서 보편적 교훈을 끌어내는 것"
예시: "유머 — 진지한 주제도 가볍게 풀어내는 것"
```

## 금기 사항 (절대 하지 않을 것)

```
[여기에 작성]
예시: "독자를 깔보는 듯한 말투"
예시: "근거 없는 과장이나 미사여구"
예시: "정보만 나열하고 해석 없이 끝내는 글"
```

---

## 마무리 스타일

글을 어떻게 끝내고 싶은가?

```
[여기에 작성]
예시: "독자에게 질문을 던지며 끝내기 — 생각거리를 남긴다"
예시: "핵심 한 문장으로 압축해서 끝내기 — 여운을 남긴다"
예시: "다음 행동을 제안하며 끝내기 — 실행으로 연결한다"
```

## 종결어미

```
[여기에 작성]
예시: "~합니다체 (기본)"
예시: "~다체 (에세이, 칼럼)"

```

## 채널별 톤 (선택사항)

채널마다 다른 톤을 원한다면 적어주세요. 안 쓰면 위의 기본 어조가 모든 채널에 적용됩니다.

```
[여기에 작성 — 선택사항]
예시:
- 블로그: 전문적이고 분석적
- 브런치: 개인적이고 성찰적
- SNS: 짧고 위트 있게
- 보고서: 객관적이고 데이터 중심
```

## 롤모델 글쓰기 (선택사항)

좋아하는 작가나 글쓰기 스타일이 있다면?

```
[여기에 작성 — 선택사항]
예시: "유시민 — 논리적이면서도 읽기 쉬운 문장"
예시: "김영하 — 짧은 문장, 강한 인상"
예시: "특정 블로거나 유튜버의 스타일도 OK"
```

