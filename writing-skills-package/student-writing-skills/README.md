# 나만의 글쓰기 스킬 패키지 ✍️

> AI 시대, 뛰어난 재능이나 지능이 부족해도 AI를 세컨드 브레인으로
> **'어떤 방향으로 활용하느냐'**가 중요합니다.
> 이 스킬 패키지는 여러분만의 글쓰기 스타일을 AI에 학습시켜,
> 매번 일관된 품질의 글을 쓸 수 있게 도와줍니다.

---

## 📦 패키지 구성

```
student-writing-skills/
├── README.md                          ← 지금 읽고 있는 파일
├── my-style.md                        ← ⭐ 나만의 스타일 설정 파일
├── writing-workflow/
│   └── SKILL.md                       ← 일반 글쓰기 (뉴스레터, 칼럼, 보고서 등)
├── blog-writing-workflow/
│   └── SKILL.md                       ← 블로그 글쓰기 (워드프레스 SEO 최적화)
└── brunch-writing-workflow/
    └── SKILL.md                       ← 브런치 글쓰기 (큐레이션 최적화)
```

## 🚀 시작하기 (3단계)

### Step 1. 나만의 스타일 설정하기

`my-style.md` 파일을 열어서 `[여기에 작성]` 부분을 채워넣으세요.

어렵게 생각하지 마세요! 아래처럼 편하게 쓰면 됩니다:

```
나는 친근하고 유머러스한 톤으로 글을 쓰고 싶어.
전문 용어는 가급적 쉽게 풀어쓰고,
독자와 대화하듯 글을 쓰는 게 내 스타일이야.
```

또는 좀 더 구체적으로:

```
어조: 친근한 선배처럼, 가르치지 않고 함께 배우는 느낌
특기: 비유와 예시를 많이 활용
금기: 딱딱한 논문체, "~할 수 있다" 식의 모호한 표현
좋아하는 문장: 짧고 임팩트 있는 문장
마무리 스타일: 독자에게 질문을 던지며 끝내기
```

### Step 2. Claude Code에 업로드하기

1. Claude Code 프로젝트의 `/mnt/skills/user/` 폴더에 각 스킬 폴더를 업로드합니다
2. `my-style.md`는 각 스킬 폴더 안에 함께 넣어주세요

```
/mnt/skills/user/
├── writing-workflow/
│   ├── SKILL.md
│   └── my-style.md
├── blog-writing-workflow/
│   ├── SKILL.md
│   └── my-style.md
└── brunch-writing-workflow/
    ├── SKILL.md
    └── my-style.md
```

### Step 3. 바로 사용하기

이제 Claude에게 "글 써줘", "블로그 글 써줘", "브런치 글 써줘"라고 말하면
여러분의 스타일이 반영된 글쓰기 워크플로우가 자동으로 시작됩니다!

## 💡 활용 팁

### "나만의 스타일을 아직 모르겠어요"

괜찮습니다! 아래 방법을 시도해보세요:

1. **좋아하는 글 3편을 찾아보세요** — 그 글들의 공통점이 여러분의 스타일입니다
2. **싫어하는 글의 특징을 적어보세요** — "이런 건 싫다"가 스타일의 출발점입니다
3. **Claude에게 물어보세요** — "내 글 스타일을 분석해줘"라고 기존 글을 보내면 분석해줍니다

### 스타일은 언제든 바꿀 수 있어요

`my-style.md`를 수정하면 다음 글부터 새 스타일이 적용됩니다.
여러 버전을 만들어서 상황에 따라 바꿔쓰는 것도 좋은 방법입니다.

### 채널별로 톤을 다르게 설정하고 싶다면

`my-style.md`에 채널별 톤을 따로 적어주면 됩니다:

```
## 채널별 톤
- 블로그: 전문적이고 분석적 (검색 유입 고려)
- 브런치: 개인적이고 감성적 (큐레이션 고려)
- SNS: 짧고 임팩트 있게 (스크롤 멈추게)
```

## 📝 스킬별 사용법

### 일반 글쓰기 (writing-workflow)

**이럴 때 사용하세요:** 뉴스레터, 칼럼, 보고서, 기고문, 강의 스크립트, SNS 롱폼 등

**사용 예시:**
- "뉴스레터 글 써줘"
- "AI 교육에 대한 칼럼 써줘"
- "프로젝트 보고서 작성해줘"

### 블로그 글쓰기 (blog-writing-workflow)

**이럴 때 사용하세요:** 워드프레스 블로그, 네이버 블로그 등 SEO 검색 최적화가 필요한 글

**사용 예시:**
- "블로그 글 써줘"
- "SEO 최적화된 블로그 포스트 만들어줘"

### 브런치 글쓰기 (brunch-writing-workflow)

**이럴 때 사용하세요:** 브런치스토리에 발행할 글

**사용 예시:**
- "브런치 글 써줘"
- "브런치에 올릴 에세이 써줘"

## ❓ FAQ

**Q. 프로그래밍을 몰라도 쓸 수 있나요?**
A. 네! 파일을 폴더에 넣고, Claude에게 말만 하면 됩니다.

**Q. my-style.md를 안 채우면 어떻게 되나요?**
A. 기본 스타일(차분하고 전문적인 설명체)로 작동합니다. 하지만 꼭 채워넣는 걸 추천합니다!

**Q. 글쓰기 5단계를 꼭 다 거쳐야 하나요?**
A. 아닙니다. 이미 초안이 있으면 "리라이팅해줘"라고만 하면 됩니다.
"한 번에 해줘"라고 하면 5단계를 자동으로 쭉 진행합니다.

**Q. 스킬 3개를 다 써야 하나요?**
A. 아닙니다. 필요한 채널의 스킬만 업로드하면 됩니다.
